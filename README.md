# Handwritten-Digits-Fall-2025

Handwritten digit image collection for CSCI 580 Fall 2025

Following these for your submission:

Your images should be uploaded to the folder /digits/

Use this filename convention:

      <label/digit>-<groupID>-<memberID>.png

For example, image of digit 3 by Group 4 member 1, the filename is 3-4-1.png
